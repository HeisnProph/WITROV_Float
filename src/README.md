# This project was designed for the 2026 mateRov float tasks


## Board
- Arduino Nano ESP32

## task
- Descending to 2.5m
- Ascending to 0.4m
- Descending to 2.5m again
- Ascending to surface
- Record data all the time and export profiling for this mission

## parameters
- Parameters can be adjusted for future tasks in the float_webserver.cpp

## Build
- Copy every file and open float_project.ino in Arduino IDE. Compile and upload to board.
